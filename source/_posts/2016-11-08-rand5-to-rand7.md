问题：已知rand5生成rand7
<!-- more -->
可以证明：
若
```
randX->[0, X-1]
```
则
```
X*randX + randX -> [0, X^2 - 1]
```
那么
``` java
int rand7() {
    int r = 0;
    while (true) {
        r = 5*rand5() + rand5();
        if (r < 21) {
            break;
        }
    }
    return r % 7;
}
```
